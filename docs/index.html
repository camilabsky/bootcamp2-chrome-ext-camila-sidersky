Documentação da Extensão Murloc

Bem-vindo à documentação da Extensão Murloc! Esta extensão para Chrome traz o charme dos Murlocs para seu navegador.

Instalação

  1. Baixe a extensão
  2. Abra o Chrome e acesse chrome://extensions/
  3. Ative o Modo Desenvolvedor
  4. Clique em "Carregar sem compactação" e selecione a pasta da extensão

Funcionalidades

  • Interface popup com tema Murloc
  • Botão interativo com respostas do Murloc
  • Service worker em segundo plano para tratamento de mensagens

Uso

Clique no ícone da extensão na barra de ferramentas do navegador para abrir o popup do Murloc. Clique no botão para ouvir uma saudação do Murloc!
